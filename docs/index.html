What is Trachoma Finder?

How do I install Trachoma Finder?

If you have MATLAB:

  1. Download latest version from the top of this page (icon with matlab logo)
  2. Install Trachoma Finder using MATLAB's "Install App" button
  3. Start Trachome Finder from your Apps list.
  4. Head to the user manual to learn about all functionalities

If you do not have MATLAB:

  1. Download latest version for your OS from the top of this page
  2. Install Trachoma Finder double-clicking the downloaded file
  3. Start Trachoma Finder from your Apps list.
  4. Head to the user manual to learn about all functionalities
